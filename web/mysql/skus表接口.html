# skus[商品规格表] 所有接口

# 一、商品规格列表

# ① thinkphp框架接口

点击查看具体文档：【三、商品规格获取列表、新增、修改、修改状态、删除接口】

  1. 请求方式：GET [用postman测试]
  2. 接口示例：
    本地路由地址：http://thinkphp.shop/admin/skus/:page?limit=[:limit]&order=[:order]&keyword=[:keyword]
    本地路由示例：http://thinkphp.shop/admin/skus/1?limit=20&order=asc&keyword=尺寸 (opens new window)
    1代表页码, limit代表每页多少条, keyword代表搜索关键字，搜商品规格名称name字段, order代表排序规则
    limit、keyword、order 是可选参数，limit默认为 20，keyword默认为 空字符串，order 默认为 desc
  3. header头
参数 是否必填 类型 说明
token 是 String token值
  1. 请求参数
参数 是否必填 类型 说明
page 是 int 页码，默认1
limit 否 int 每页多少条数据，选填，默认20
order 否 String 排序规则，选填，默认desc
keyword 否 String 搜索关键字，选填，搜商品规格名称name字段，默认为 空字符串
  1. 返回
{
    "msg": "ok",
    "data": {
        "totalCount": 2,
        "list": [
            {
                "id": 4,
                "type": 3,
                "name": "尺寸",
                "default": "xl,xxl,xxxl,sl,xsl",
                "order": 50,
                "status": 1,
                "create_time": "2025-04-24 12:36:12",
                "update_time": "2025-04-24 12:36:12"
            }
        ]
    }
}

status 否 int 可用状态，0不可用，1可用，可不填，默认：1

点击查看具体文档：【四. 批量删除商品规格】

}
