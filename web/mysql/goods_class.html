      # 商品管理板块表：
      goods_class[商品分类表]、skus[商品规格表]、goods[商品表] 说明

      # 一、goods_class[商品分类表] 字段设计

      字段名 数据类型及描述 空

      默认值

      字段含义

      id int(20)
      主键、自增长、UNSIGNED无符号       		否 主键id
      pid int(20) 否 0 上一级(父级)id
      name varchar(30) 否 商品分类名称
      desc varchar(255) 是 商品分类描述，对商品分类的简单介绍
      order int(11) 是 50 排序，默认50
      status int(1) 否 1 可用状态：0禁用1启用
      create_time datetime 否 CURRENT_TIMESTAMP 数据创建时间
      update_time datetime 否 CURRENT_TIMESTAMP 数据更新时间

      额外说明：mysql每行最大只能存65535个字节。假设是utf-8编码，每个字符占3个字节。varchar存储最大字符数为(65535-2-1)/3=21844字符长度

      ...
      sku_value text 是 商品规格属性（商品参数）
      goods_tags varchar(2000) 是 商品标签（如：近期销量飙升，30天上新，近7天同类热卖，24小时内发货，等等，用逗号隔开）
      ...
      order int(11) 是 50 排序，默认50
      status int(1) 否 1 可用状态：0禁用、1启用
      create_time datetime 否 CURRENT_TIMESTAMP 数据创建时间
      update_time datetime 否 CURRENT_TIMESTAMP 数据更新时间
