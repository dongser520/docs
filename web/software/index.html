    # Egg.js [Egg.js官网]

    Egg.js是阿里提供的Node.js应用的核心基础设施，它面向的领域是：企业级的 web 基础框架。
    是在基于Nodejs的框架：Express和Koa框架的基础上（Express和Koa缺乏约定，缺乏规范）
    Eggjs对MVA进行约定规范，Eggjs为企业级框架和应用而生。
    使用前记得在vscode中先安装eggjs的插件：eggjs/Egg.js dev tools等
