# 第四学期第一季课程介绍

# 课程名称

《thinkphp / egg.js + mysql 开发商城系统功能后台及API》

# 课程介绍

本季课程，我们将同时采用 thinkphp + mysql, egg.js + mysql 两套技术栈框架来开发我们商城系统的后台和api接口部分。以满足不同技术栈的同学，在学习我们移动端app、小程序、H5端，关于商城部分的后端开发和API接口的开发。

# 适用人群

  1. 学习过我们第二学期第三季课程【即学习过我们企业网站前后端完整开发】的同学，可以来学习本季课程；
  2. 了解node.js基础知识，或者使用过egg.js框架的同学，可以来学习本季课程；
  3. 有php开发的经历，了解php基本语法，可以尝试学习本套课程；
  4. 用thinkphp框架开发过项目，可以来学习本套课程；
