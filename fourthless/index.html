# 第四学期课程总述

第四学期主要是为了满足以下同学设计的课程：

  1. 想继续学习第三学期移动端，掌握后端api如何开发的同学（我们第三期移动端课程，大部分提供的都是线上api，有些同学想学习后端api开发）；
  2. 想进一步学习全栈开发的同学，即前后端都想自己做，来开发项目；
  3. 有一定的技术栈，比如：你有php的基础，习惯于用thinkphp开发后端，或者你有node.js基础，并且学过我们前面的egg.js基础，想继续用egg.js开发后端等等；
  4. 想学习更多技术栈知识，如：vue2/vue3，vite，深入学习MySql数据库，更多前端框架或组件库如：elementUI、Element Plus等等；
    如果你有以上中的一种或者多种需求，可以来学习我们第四学期课程。
